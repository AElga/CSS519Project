The four main areas that I think have the metrics that I need from the project are:

# API Logic

This includes:

- Whether API calls are succeeding or failing
- Latency/delay of Responses
- If APIs are reliant on a Third party service, whether those services are online or not

# Performance

This includes:

- CPU/RAM usage
- Network health

# Database

This includes:

- Read/Update/Delete rates
- Disk size (storage limits)

# Security

This includes:

- Login Attempts
- Network Traffic Spikes
- Anomalous events/logs on restricted services
